Orbit Diagram + Formulas

  • Gravitational force: F = GMm / r²
  • Orbital velocity: v₀ = √(GM / r)
  • Escape velocity: vₑ = √(2GM / r)
  • Time period: T = 2πr / v₀
  • Gravity at height: g' = GM / r²
  • Stable orbit occurs when gravity provides required centripetal force.

Simple Explanation

A satellite stays in orbit because gravity continuously pulls it toward the planet and provides the centripetal force. If speed is too low, it falls toward the planet. If speed is close to orbital velocity, it stays in stable orbit. If speed reaches escape velocity, it escapes the planet’s gravity.

NEET Traps

  • Satellite is always falling toward Earth, but its tangential speed keeps it orbiting.
  • Orbital velocity decreases as orbital radius increases.
  • Escape velocity is √2 times orbital velocity at the same radius.
  • Orbital velocity does not depend on satellite mass.
  • Gravity at height is less than gravity at planet surface.
