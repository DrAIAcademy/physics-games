Logic Gate Diagram + Truth Rules

  • AND gate: Y = A·B, output 1 only when both inputs are 1.
  • OR gate: Y = A + B, output 1 when at least one input is 1.
  • NOT gate: Y = A̅, output is opposite of input A.
  • NAND gate: Y = A·B̅ style idea, opposite of AND: Y = (A·B)'
  • NOR gate: opposite of OR: Y = (A + B)'
  • XOR gate: output 1 when inputs are different.

Simple Explanation

Logic gates are electronic circuits that give output based on input signals 0 and 1. In NEET, you must know the truth table and output behavior of AND, OR, NOT, NAND and NOR gates. This game trains you to identify the correct gate quickly.

NEET Traps

  • NAND is the opposite of AND, not the same as OR.
  • NOR is the opposite of OR.
  • NOT gate has only one input.
  • AND output is 1 only for A = 1 and B = 1.
  • OR output is 0 only for A = 0 and B = 0.

Observation Questions
