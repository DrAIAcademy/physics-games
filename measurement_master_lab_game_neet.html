Formula Focus

  • Vernier reading = MSR + VSD × LC
  • Screw gauge reading = MSR + CSD × LC
  • Corrected reading = observed reading − zero error
  • Zero correction = − zero error
  • Positive zero error is subtracted.
  • Negative zero error is added.
  • Least count = value of one smallest division.

Simple NEET Explanation

Vernier caliper and screw gauge are used for precise measurement. First note the main scale reading, then add the coinciding division multiplied by least count. Finally apply zero correction.

In this game, your lab unlocks when the corrected reading matches the actual target size.

NEET Traps

  • Do not forget zero error correction.
  • Positive zero error is subtracted from observed reading.
  • Negative zero error is added to observed reading.
  • Least count must be in same unit as main scale reading.
  • Main scale reading is taken before zero of vernier/circular scale.
  • Coinciding division is multiplied by least count.
