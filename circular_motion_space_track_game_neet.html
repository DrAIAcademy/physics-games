Circular Motion Diagram + Formulas

  • Centripetal acceleration: a = v² / r
  • Centripetal force: F = mv² / r
  • Angular velocity: ω = v / r
  • Time period: T = 2πr / v
  • Maximum safe speed on level track: vmax = √(μrg)
  • If v > vmax, friction is not enough and object skids outward.

Simple Explanation

In circular motion, velocity direction changes continuously, so acceleration is present even if speed is constant. On a level circular track, friction provides the required centripetal force. If speed is too high, friction cannot provide enough inward force and the spacecraft skids outward.

NEET Traps

  • Centripetal force is inward, not outward.
  • Constant speed does not mean zero acceleration.
  • On a level road/track, friction provides centripetal force.
  • Maximum safe speed is independent of mass because mass cancels.
  • If the string breaks, the object moves tangentially, not radially outward.
